# Bank_Credit_churn_predict
Bank Mijozlarining ma'lumotlarini tahlil qilib kreditni qaytara olish ehtimolini aniqlovchi Machine Learning model yaratish

Ushbu model har bir bank mijozining kreditni qaytara olish ehtimolligiga yuqori ta'sir qiluvchi omillarni hisobga olib, ma'lumotlarini tahlil qilib, har bir mijoz uchun 'id' raqami va uning kreditni qaytara olish ehtimolligini natijalar sifatida taqdim qiladi.
